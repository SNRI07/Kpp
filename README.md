`long` `긴`\
`long long`=`긴긴`\

`cout`=`출력`\
`cin`=`입력`\
`return`=`반환`\
`main`=`진입`\
`for`=`반복`\
`while`=`동안`\
`if`=`만약`\
`else`=`아니면`\
`brake`=`멈춰`\
`continue`=`계속`\
`string`=`문자열`\
`sort`=`정렬`\

`최대값`=`max_element`
