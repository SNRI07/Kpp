# 변수
`int`=`정수`\
`short`=`짧은`\
`long` `긴`\
`long long`=`매우긴`\
`float`=`뜨다`\
`char`=`숯`
# 명령어
`cout`=`출력`\
`cin`=`입력`\
`return`=`반환`\
`main`=`진입`\
`for`=`반복`\
`while`=`동안`\
`if`=`만약`\
`else`=`아니면`\
`break`=`멈춰`\
`continue`=`계속`\
`string`=`문자열`\
`sort`=`정렬`\
`unique`=`중복제거`\
`find`=`찾기`\
`min_element`=`최소값`\
`max_element`=`최대값`
